# Kitchen Echoes

A no-build, browser-based sensory memory game. The central kitchen connects four fantasy spaces: the Living Volcano, Wishing Well, Forest, and Machine Works.

## Publish on GitHub Pages

1. Create a GitHub repository.
2. Upload every file in this folder to the repository root.
3. Go to **Settings → Pages**.
4. Choose **Deploy from a branch**, select `main` and `/ (root)`, then save.

No installation or build command is required. Open `index.html` to test locally.

## Editing content

Character stories, worlds, clues, colours, and objects are in the `people` array at the top of `app.js`.
